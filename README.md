# TUUG — Growth Systems Website

Performance-driven digital marketing agency website built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM
- TanStack Query

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
